# 广外梁同学の文件幻变小窝

A multi-format file converter web app (docx/pdf/excel/image) with Chinese/English UI, built with Flask.

## 功能 Features
- DOCX ⇄ PDF
- Excel ⇄ PDF
- 图片合并转PDF（纵向拼接）
- PDF转Word/Excel/图片/长图
- 支持中英文界面切换
- 全平台本地/云端部署

## 使用方法 Usage
1. 上传文件，选择目标格式，点击转换。
2. 支持批量图片合成长图PDF。
3. 转换结果自动下载。
